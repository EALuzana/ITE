In this project, the files' names are arranged with a pattern.

use prefix `_` when naming or using an image file:
```
"_brain.png"
"_pic.jpg"
"_new image.jpg"
```

css files should have `$` in the file's 1st name:
```
"$nav.css"
"$theme.html"
"$style-a.html"
```
